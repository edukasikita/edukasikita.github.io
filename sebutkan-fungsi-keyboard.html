8. Sebutkan dan jelaskan fungsi tombol-tombol pada keyboard yang anda ketahui


1. Fungsi Tombol Esc Tombol Escape untuk membatalkan perintah yang sudah dilakukkan. Jika kita menekan tombol ini maka perintah atau dialog yang kita masukkan akan dibatalkan.
2. Fungsi Tombol Tab Tombol Tab berfungsi jika ditekan akan memindahkan cursor ke arah kanan dengan satu tabulansi.
3. Fungsi Tombol Caps Lock  Jika menekan tombol ini maka lampu indikator caps lock pada keyboard menyala, hal ini menandakan bahwa anda menggunakan mode semua huruf besar atau semua huruf kapital, jika anda menekan lagi dan lampu indikator mati maka dalam mode seperti biasa atau semua huruf kecil.
4. Fungsi Tombol Shift Fungsinya yaitu untuk membuat huruf besar secara manual (dengan menekan shift+a = A), dan untuk memasukkan input yang diatas angka 1,2,3 dan seterusnya (contoh : shift+1 = ! , shift+/ = ?).
5. Fungsi Tombol Ctrl Khusus untuk tombol ini fungsinya itu untuk menyisipkan perintah-perintah pintas. Contohnya untuk mencopy  file tinggal tekan ctrl+c.
6. Fungsi Tombol Alt Fungsinya sama seperti tombol Ctrl yaitu untuk menyisipkan perintah secara pintas. Contohnya alt+F4, itu untuk menutup program yang sedang dijalankan.
7. Fungsi Tombol Enter Sebagai perintah "ok" yang diberikan oleh program pada komputer dan untuk membuat paragraf baru.
8. Fungsi Tombol Back Space  Fungsinya untuk mendelete satu huruf yang berada disebelah kiri kursor. Jika kita kombinasikan dengan tombol ctrl, ctrl+backspace maka anda menghapus satu kata disebelah kiri kursor.
9. Tombol Delete Tombol ini sama seperti tombol backspace, bedanya itu jika tombol delete menghapus satu huruf di sebelah kanan kursor.

22. Sebutkan 15 tombol dan fungsi pada keyboard


1. Fungsi Tombol Esc
Tombol Escape untuk membatalkan perintah yang sudah dilakukkan. Jika kita menekan tombol ini maka perintah atau dialog yang kita masukkan akan dibatalkan.

2. Fungsi Tombol Tab
Tombol Tab berfungsi jika ditekan akan memindahkan cursor ke arah kanan dengan satu tabulansi.

3. Fungsi Tombol Caps Lock 
Jika menekan tombol ini maka lampu indikator caps lock pada keyboard menyala, hal ini menandakan bahwa anda menggunakan mode semua huruf besar atau semua huruf kapital, jika anda menekan lagi dan lampu indikator mati maka dalam mode seperti biasa atau semua huruf kecil.

4. Fungsi Tombol Shift
Fungsinya yaitu untuk membuat huruf besar secara manual (dengan menekan shift+a = A), dan untuk memasukkan input yang diatas angka 1,2,3 dan seterusnya (contoh : shift+1 = ! , shift+/ = ?).

5. Fungsi Tombol Ctrl
Khusus untuk tombol ini fungsinya itu untuk menyisipkan perintah-perintah pintas. Contohnya untuk mencopy  file tinggal tekan ctrl+c.

6. Fungsi Tombol Alt
Fungsinya sama seperti tombol Ctrl yaitu untuk menyisipkan perintah secara pintas. Contohnya alt+F4, itu untuk menutup program yang sedang dijalankan.

7. Fungsi Tombol Enter
Sebagai perintah "ok" yang diberikan oleh program pada komputer dan untuk membuat paragraf baru.

8. Fungsi Tombol Back Space 
Fungsinya untuk mendelete satu huruf yang berada disebelah kiri kursor. Jika kita kombinasikan dengan tombol ctrl, ctrl+backspace maka anda menghapus satu kata disebelah kiri kursor.

9. Tombol Delete
Tombol ini sama seperti tombol backspace, bedanya itu jika tombol delete menghapus satu huruf di sebelah kanan kursor.

10. Tombol Macro 
Macro ini jika ditekan akan muncul seperti ini "\".

11. Tombol Insert 
Untuk menyisipkan karakter di posisi kursor.

12. Tombol Home
Fungsinya untuk langsung menuju bagian atas halaman atau lembar kerja.

13. Tombol End
Fungsinya untuk langsung menuju bagian bawah halaman atau lembar kerja.

14. Tombol Page Up
Fungsinya untuk menggerakkan baris satu demi satu layar ke atas.

15. Tombol Page Down
Fungsinya untuk menggerakkan baris satu demi satu layar ke bawah.

Penjelasan:

Tombol Backspace

Tombol Backspace terletak di sisi kanan atas keyboard. Tombol ini berfungsi untuk menghapus satu karakter yang ada di depan kursor saat ada terdapat kesalahan saat mengetik huruf. Tekan tombol Backspace dan tombol Ctrl bersamaan untuk menghapus satu kata di depan kursor.

Tombol Delete

Bila tombol Backspace dapat digunakan untuk menghapus satu karakter di depan kursor, maka tombol Delete dapat digunakan untuk menghapus satu karakter di belakang kursor. Tombol ini juga terletak di sisi kanan atas. Jika Anda menekan tombol Delete dan tombol Ctrl bersamaan, maka Anda dapat menghapus satu kata sekaligus.

Tombol Caps Lock

Terletak di sisi kiri, tombol Caps Lock berfungsi untuk membuat tulisan menjadi huruf kapital. Akan ada lampu menyala pada tombol tersebut sebagai tanda bahwa Caps Lock sedang aktif.

Tombol Tab

Berada di atas tombol Caps Lock, tombol Tab dapat digunakan untuk memindahkan kursor dalam satu jeda ke arah kanan. Bila Anda menekan tombol keyboard ini, maka kursor akan bergeser ke kanan dengan jarak satu tabulasi. Anda pun jadi tidak perlu menekan tombol spasi berulang kali.

Tombol Esc

Tombol Esc bisa Anda temukan di ujung kiri atas pada keyboard. Fungsinya adalah untuk membatalkan perintah yang telah Anda lakukan. Jika Anda tidak ingin melanjutkan perintah yang sudah Anda berikan, maka Anda bisa menekan tombol ini untuk membatalkannya.

Tombol Home

Jika Anda ingin memindahkan kursor ke ujung kiri dengan cepat pada satu baris saat mengetik, maka Anda bisa menekan tombol Home. Tombol ini biasanya jadi satu dengan tombol panah kiri (?). Agar bisa berfungsi, Anda harus menekannya bersamaan dengan tombol fn. Jika Anda menekan tombol Home dan tombol Ctrl bersamaan, maka kursor akan bergeser ke sisi kiri pada baris pertama paling atas.

Tombol End

Fungsi tombol End berkebalikan dengan tombol Home. Tombol keyboard ini berfungsi untuk memindahkan kursor ke sisi kanan akhir dalam satu baris tulisan Anda. Tombol End bisa ditemukan pada tombol panah kanan (?). Sama seperti tombol Home, Anda harus menekannya bersamaan dengan tombol fn untuk membuatnya berfungsi. Tekan tombol End dan tombol Ctrl bersamaan untuk memindahkan kursor ke sisi kanan paling akhir.

Tombol Enter

Tombol Enter merupakan tombol yang penting pada keyboard dan memiliki beberapa fungsi. Tombol keyboard ini dapat digunakan untuk memindahkan kursor ke ke baris di bawahnya saat Anda mengetik. Fungsi lainnya adalah saat sebagai perintah saat Anda menyetujui suatu program yang ada pada komputer atau laptop Anda.

Tombol Page Up dan Page Down

Tombol Page Up berfungsi untuk menggerakkan baris satu layar ke arah atas, sedangkan Page Down kebalikannya, yakni menggeser satu baris ke bawah. Intinya, kedua tombol tersebut akan menggeser kursor ke halaman atas atau bawah.
