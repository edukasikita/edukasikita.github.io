13. data yang bertipe teks seting disebut dengan istilah


Jawaban :

Data yang bertipe teks seting (apakah yang anda maksud "string" ya?) disebut dengan istilah “array of char”

Penjelasan :

Tipe data adalah jenis data yang dapat diolah oleh komputer untuk memenuhi kebutuhan dalam pemrograman komputer.

Ada banyak tipe data yang tersedia, tergantung jenis bahasa pemrograman yang dipakai. Namun secara umum dapat dikelompokkan menjadi dua :

1. Tipe data primitive - ini adalah tipe data dasar yang tersedia secara langsung pada suatu bahasa pemrograman.

2. Tipe data composite - ini adalah tipe data bentukan yang terdiri dari dua atau lebih tipe data primitive.

Tipe data yang sering dipakai oleh pascal, meliputi

1. Integer (bilangan bulat)

adalah tipe data yang terdiri dari angka bulat (tidak mengandung nilai pecahan atau nilai desimal).

2. Real (bilangan pecahan)

adalah tipe data yang terdiri dari angka bulat (tidak mengandung nilai pecahan atau nilai desimal).

3. Char (alphanumerik dan tanda baca)

adalah tipe data yang hanya bisa diisi dengan 1 karakter saja. Karakter ini (alpanumerik) harus ditulis diantara tanda kutip.  

Karena hanya bisa diisi dengan 1 karakter saja, tipe data char hanya membutuhkan 1 byte memory (1 byte = 8 bit).

4. Boolean (logika)

merupakan tipe data logika, yang berisi dua kemungkinan nilai: TRUE (benar) atau FALSE (salah).

Tipe data boolean banyak digunakan untuk percabangan kode program, atau untuk memutuskan apa yang mesti dijalankan ketika sebuah kondisi terjadi.

5. Tipe Data String

Sebenarnya string termasuk tipe data array dan bukan termasuk tipe data primitive.

Jadi string merupakan data yang menyimpan array dari suatu karakter, sebagai contoh 'ABCDEF' merupakan konstanta string yang berisikan 6 byte karakter. Ukuran Tempat untuk tipe data ini adalah 2 s/d 256 byte, dengan jumlah elemen 1 s/d 255. String dideklarasikan dengan string [ konstanta ] atau string.

Bila ukuran string tidak didefinisikan maka akan banyak memakan ruang, karena ukuran string menyesuaikan dengan defaultnya.

38. Sebutkan macam-macam tipe data​


Jawaban:

Secara umum, tipe data dapat dibagi menjadi dua jenis, yaitu tipe data dasar dan tipe data turunan.

Tipe data dasar adalah tipe data yang tidak diturunkan dari tipe data lain. Tipe data dasar terdiri dari:

Integer: Tipe data yang digunakan untuk menyimpan bilangan bulat.

Float: Tipe data yang digunakan untuk menyimpan bilangan desimal.

String: Tipe data yang digunakan untuk menyimpan teks.

Boolean: Tipe data yang digunakan untuk menyimpan nilai benar atau salah.

Date: Tipe data yang digunakan untuk menyimpan tanggal dan waktu.

Time: Tipe data yang digunakan untuk menyimpan waktu.

Tipe data turunan adalah tipe data yang diturunkan dari tipe data dasar. Tipe data turunan terdiri dari:

Array: Tipe data yang digunakan untuk menyimpan kumpulan data sejenis.

Object: Tipe data yang digunakan untuk menyimpan kumpulan data yang heterogen.

Enum: Tipe data yang digunakan untuk menyimpan kumpulan data yang terbatas.

Structure: Tipe data yang digunakan untuk menyimpan kumpulan data yang saling terkait.

Class: Tipe data yang digunakan untuk membuat objek.

Berikut adalah penjelasan lebih lanjut tentang masing-masing tipe data:

Integer

Tipe data integer digunakan untuk menyimpan bilangan bulat. Tipe data integer memiliki berbagai macam ukuran, yang ditentukan oleh lebar bit yang digunakan. Ukuran tipe data integer yang umum adalah:

Byte: Tipe data integer dengan lebar bit 8.

Short: Tipe data integer dengan lebar bit 16.

Int: Tipe data integer dengan lebar bit 32.

Long: Tipe data integer dengan lebar bit 64.

Float

Tipe data float digunakan untuk menyimpan bilangan desimal. Tipe data float memiliki berbagai macam ukuran, yang ditentukan oleh jumlah digit desimal yang dapat disimpan. Ukuran tipe data float yang umum adalah:

Float: Tipe data float dengan 6 digit desimal.

Double: Tipe data float dengan 15 digit desimal.

String

Tipe data string digunakan untuk menyimpan teks. Tipe data string dapat menyimpan berbagai macam karakter, termasuk huruf, angka, simbol, dan spasi.

Boolean

Tipe data boolean digunakan untuk menyimpan nilai benar atau salah.

Date

Tipe data date digunakan untuk menyimpan tanggal dan waktu. Tipe data date dapat menyimpan tanggal dan waktu dalam berbagai format.

Time

Tipe data time digunakan untuk menyimpan waktu. Tipe data time dapat menyimpan waktu dalam berbagai format.

Array

Tipe data array digunakan untuk menyimpan kumpulan data sejenis. Array memiliki ukuran yang tetap, dan setiap elemen array memiliki tipe data yang sama.

Object

Tipe data object digunakan untuk menyimpan kumpulan data yang heterogen. Object dapat terdiri dari berbagai macam tipe data, termasuk tipe data dasar dan tipe data turunan.

Enum

Tipe data enum digunakan untuk menyimpan kumpulan data yang terbatas. Enum hanya dapat menyimpan nilai-nilai yang telah ditentukan sebelumnya.

Structure

Tipe data structure digunakan untuk menyimpan kumpulan data yang saling terkait. Structure dapat digunakan untuk menyimpan data yang terdiri dari berbagai macam tipe data.

Class

Tipe data class digunakan untuk membuat objek. Class adalah blueprint yang digunakan untuk membuat objek. Objek adalah entitas yang memiliki data dan perilaku.
