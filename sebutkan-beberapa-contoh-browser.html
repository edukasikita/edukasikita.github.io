Jawaban:

1. **Web Browser** adalah perangkat lunak yang digunakan untuk menjelajahi World Wide Web (WWW) atau internet. Web browser memungkinkan pengguna untuk mengakses berbagai jenis konten di internet, termasuk halaman web, gambar, video, dan berbagai aplikasi web. Beberapa web browser terkenal meliputi Google Chrome, Mozilla Firefox, Microsoft Edge, dan lainnya.

2. **Fungsi Utama Web Browser**:

  - **Menampilkan Halaman Web**: Web browser mengambil dan menampilkan halaman web dalam format yang dapat dibaca oleh pengguna.

  - **Navigasi**: Web browser memungkinkan pengguna untuk mengklik tautan (link) dan menuju ke halaman web lain.

  - **Mengelola Riwayat**: Web browser menyimpan riwayat aktivitas pengguna, memungkinkan mereka untuk kembali ke halaman web yang telah dikunjungi sebelumnya.

  - **Menyimpan Bookmark**: Pengguna dapat menyimpan tautan favorit mereka dengan fitur bookmark, sehingga mudah diakses kembali.

  - **Penyelidikan Kata atau Frasa**: Banyak web browser memiliki kotak pencarian yang memungkinkan pengguna mencari informasi di mesin pencari.

  - **Pengelolaan Data Pengguna**: Web browser memungkinkan pengguna mengelola kata sandi, riwayat, cookie, dan preferensi lainnya.

 

3. **Manfaat Web Browser dalam Aktivitas Berinternet**:

  - Mengakses informasi di internet.

  - Berkomunikasi melalui email dan media sosial.

  - Berbelanja online.

  - Streaming video dan audio.

  - Bermain game online.

  - Menyelidiki topik tertentu melalui pencarian online.

4. **Perbedaan antara Web Browser dan Mesin Pencari**:

  - **Web Browser**: Web browser adalah perangkat lunak yang digunakan untuk mengakses dan menampilkan konten web. Itu adalah "pintu" untuk menjelajahi internet.

  - **Mesin Pencari (Search Engine)**: Mesin pencari adalah alat yang digunakan untuk mencari informasi di internet dengan memasukkan kata kunci atau frasa. Mesin pencari seperti Google, Bing, dan Yahoo membantu menemukan halaman web yang relevan dengan permintaan pencarian.

5. **Contoh Web Browser yang Paling Banyak Digunakan**:

  - Google Chrome

  - Mozilla Firefox

  - Microsoft Edge

  - Apple Safari

  - Opera

6. **Cara Menggunakan Web Browser dan Membuka Situs Web**:

  - Buka web browser yang diinstal di komputer atau perangkat Anda.

  - Ketik alamat URL (misalnya, "www(.)google(.)com") di bilah alamat dan tekan "Enter" atau "Return."

  - Halaman web yang sesuai akan ditampilkan. Anda dapat mengklik tautan, memasukkan pencarian, dan berinteraksi dengan situs web tersebut.

7. **Apa yang Bisa Ditampilkan dalam Sebuah Web Browser**:

  - Halaman web dengan teks, gambar, video, dan audio.

  - Aplikasi web dan permainan.

  - Dokumen PDF dan file teks.

  - Peta interaktif.

  - Aplikasi web seperti email, perbankan online, dan lainnya.

  - Perangkat lunak plugin seperti peramban PDF dan pengaya (add-ons).

  - Riwayat penelusuran, bookmark, dan berbagai alat manajemen pengguna.

15. sebutkan 3 contoh mesin pencari ( browser )​


DuckDuckGo. ... Qwant. ... Yahoo! Search.

Jawaban:

1. Bing

Bing adalah mesin pencari dengan market share terbesar setelah Google dan Baidu. Hal yang mencolok dari mesin pencari ini adalah tampilan visual yang berubah-ubah. Ketika Anda mengunjungi halaman depan Bing, Anda akan disuguhi foto beresolusi tinggi yang diambil dari berbagai tempat di dunia. Anda bisa mengubah tampilan ini dengan mengklik tanda < atau > di bagian kanan bawah.

Selain tampilan visual, Bing dilengkapi dengan berbagai fitur menarik. Salah satunya adalah tempat menyimpan data hasil pencarian. Dari halaman pencarian Anda bisa menekan tombol dengan simbol plus dan halaman yang Anda inginkan tersimpan di bagian My Saves. Fitur ini tentunya memudahkan Anda mengarsipkan halaman penting. Anda pun tak perlu menggunakan aplikasi pihak ketiga atau menyimpan URL secara manual.

Bing juga terhubung dengan Microsoft Office Online. Lewat halaman yang sama Anda bisa membuat berbagai dokumen secara online. Mulai dari Word, Excel, PowerPoint, OneNote, Sway, hingga OneDrive. Anda cukup login dengan akun Microsoft mengakses dan membuat dokumen yang diinginkan

2. DuckDuckGo

DuckDuckGo merupakan mesin pencari yang menghargai privasi Anda. Ia tidak mengumpulkan data apapun yang Anda berikan ke mesin pencari. Entah itu data soal kata kunci sampai informasi pribadi.

DuckDuckGo juga tidak menyimpan jejak pencarian yang Anda lakukan. Karena itu ia tak bisa menjual data ke pengiklan. Di saat yang bersamaan, mesin pencari ini otomatis menonaktifkan bot pelacak iklan. Dengan begini, data pribadi Anda akan tetap aman.

Meski begitu, bukan berarti hasil pencarian DuckDuckGo tidak akurat. Sebagai ganti dari data personal dan kata kunci yang tidak terekam, DuckDuckGo mengumpulkan hasil pencarian dari 100 sumber yang berbeda. Mulai dari crawler-nya sendiri, situs-situs crowd-source, Bing, Yandex, dan lainnya. Itu mengapa hasil pencarian dari mesin pencari satu ini tetap bisa diandalkan.

3. Search Encrypt

Hampir sama seperti DuckDuckGo, Search Encrypt merupakan mesin pencari yang memiliki perhatian ke privasi dan keamanan data. Ia mengombinasikan Secure Socket Layer (SSL) dengan AES-256 encryption untuk melindungi data Anda. Keduanya tak hanya memastikan data takkan dicuri oleh peretas dan bot, melainkan juga aman dari local users yang terhubung ke komputer Anda.

Di sisi lain, Search Encrypt juga menawarkan pencarian video dan maps yang aman. Untuk map, Search Encrypt menggunakan peta dari Open Street Map yang merupakan proyek kolaboratif yang dapat diedit secara bebas.

Sedangkan untuk pencarian video, mesin pencari satu ini memblokade iklan sambil melindungi data pribadi dan kata kunci pencarian yang Anda pakai. Karenanya, mungkin hasil pencarian video yang Anda temukan takkan selengkap Google. Meski begitu, fitur keamanan yang cukup lengkap menjadikan Search Encrypt sebuah browser yang patut dijajal.

Pengertian website

Website adalah kumpulan halaman dalam suatu domain yang memuat tentang berbagai informasi agar dapat dibaca dan dilihat oleh pengguna internet melalui sebuah mesin pencari. Informasi yang dapat dimuat dalam sebuah website umumnya berisi mengenai konten gambar, ilustrasi, video, dan teks untuk berbagai macam kepentingan.

Biasanya untuk tampilan awal sebuah website dapat diakses melalui halaman utama (homepage) menggunakan browser dengan menuliskan URL yang tepat. Di dalam sebuah

Sejarah website

Sejarah website pertama kali dimulai dari seorang ilmuwan yang berasal dari Inggris, bernama Tim Berners-Lee. Orang tua dari Berners juga merupakan ilmuwan komputer pada era awal dunia komputasi.

Tujuan awal dari Tim Berners membuat sebuah website adalah supaya lebih memudahkan para peneliti di tempat kerjanya untuk mendapatkan dan bertukar informasi. Kemudian, pada tanggal 30 April 1993, secara resmi CERN yang merupakan laboratorium fisika di Swiss mengumumkan tentang perilisan website secara gratis
