19. Latihan soal pts ppkn kelas 7 semester 1 kurikulum merdeka belajar.


UUD 1945 adalah salah satu peraturan perundang-undangan tertinggi yang ada di Indonesia. Berdasarkan soal, dapat disimpulkan bahwa latihan soal PTS PPKN kelas 7 semester 1 kurikulum merdeka belajar antara lain sebagai berikut:

1. Pembukaan UUD 1945 terdiri atas ... alinea.

a. 1                                  c. 3

b. 2                                 d. 4

Jawab: 4 (D).

2. Tujuan nasional dari bangsa Indonesia yang ingin dicapai melalui upaya pembangunan nasional, tercantum dalam Pembukaan UUD 1945 alinea …

a. pertama                     c. ketiga

b. kedua                        d. keempat

Jawab: keempat (D).

Pembahasan:

Setiap negara memiliki konstitusi. Konstitusi berfungsi untuk acuan dalam penyelenggaraan pemerintahan dan kehidupan masyarakat. Konstitusi yang ada di Indonesia yaitu UUD 1945. UUD 1945 adalah salah satu peraturan perundang-undangan tertinggi yang ada di Indonesia.

36. ayo kita berlatih 2.7 halaman 155 kelas 7 kurikulum 2013


Ayo kita berlatih 2.7 halaman 155 kelas 7 kurikulum 2013
Penyelesaian :
1.      Diketahui himpunan A = {1, 2, 3, 4, 5, 6, 7, 8}, himpunan B = {1,
3, 5, 7}, C = {1, 2, 3, 4}, himpunan D = {4, 5, 6, 7}, tentukan
anggota-anggota dari
a.      A irisan B
Jawab : {1, 3, 5, 7}
b.      A irisan C
Jawab : {1, 2, 3, 4}
c.      B irisan C
Jawab : {1, 3}
d.      C irisan D
Jawab :{4}
e.      B irisan D
Jawab : {5, 7}

2.      Diketahui :
A = { bilangan asli < 20}
B = {bilangan asli genap < 15}
C = {bilangan asli ganjil < 10}
D = {bilangan asli > 7 dan < 15}
Pertanyaan
a.      Tentukan anggota dari himpunan A, B, C, D
Jawab :
A = {1, 2, 3, 4, .. , 19}
B  = {2, 4, 6, 8, 10, 12, 14}
C = {1, 3, 5, 7, 9}
D  = {8, 9, 10, 11, 12, 13, 14}
b.      Tentukan anggota dari B irisan C, B irisan D, dan C irisan D.
Jawab :
B irisan C = { } atau himpunan kosong
B irisan D = { 8, 10, 12, 14}
C irisan D = {9}
c.      Diagram venn terlampir.
3.      Diketahui
S = {x | -3 <= x <= 6, x E B}
P = {x | 0 <= x <= 5, x E B}
Q = {x | -2 <= x <= 2, x E B
R = { x | -1 <= x <= 8, x E B}
Pertanyaan
a.      Tentukan anggota dari himpunan P, Q, R, dan S
Jawab :
Karena x E B artinya x anggotanya bilangan bulat.
P = {0, 1, 2, 3, 4, 5}
Q = {-2, -1, 0, 1, 2}
R = {-1, 0, 1, 2, 3, 4, 5, 6, 7, 8}
S = {-3, -2, -1, 0, 1, 2, 3, 4, 5, 6}
b.      Tentukan anggota dari P irisan Q, P irisan R, Q irisan R dan P
irisan Q irisan R.
Jawab :
P irisan Q = {0, 1, 2}
P irisan R = {0, 1, 2, 3, 4, 5}
Q irisan R = {-1, 0, 1, 2}
P irisan Q irisan R = { 0, 1, 2}
c.      Diagram venn terlampir.
