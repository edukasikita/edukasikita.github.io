Jawaban :

Chotto matte artinya "Tunggu bentar".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Simpelnya, kalimat ini terdiri dari 2 kata, yaitu Chotto (Bentar) dan Matte (Tunggu). Bila digabung akan berarti "Tunggu bentar". Kalimat ini merupakan kalimat perintah untuk menunggu sebentar secara kasual. Biasa diucapkan ke teman atau orang yang sederajat. Pada kalimat ini juga bisa ditambahkan kata Kudasai di akhir untuk membuat kalimatnya menjadi lebih sopan.

Untuk pembahasan lebih lanjut, akan saya bahas masing-masing kata nya.

1. ちょっと / Chotto

Berarti "Bentar", "Agak", "Dikit", dsb. Kata ini merupakan versi kasualnya dari kata 少し (Sukoshi) yang artinya serupa. Perbedaan lainnya yaitu kata Chotto juga bisa digunakan untuk menolak sesuatu secara sopan. Contoh penggunaannya misalkan kita ingin menolak sesuatu, dapat mengatakan 'Sumimasen, sore wa chotto...', yang secara harfiah berarti "Maaf, itu agak...".

2. 待って / まって / Matte

Berarti "Tunggu". Kata ini merupakan bentuk Te dari kata kerja 待つ (Matsu), yang berarti "Tunggu". Fungsi dari bentuk Te ini bisa untuk menjadi perintah, bisa juga untuk penyambung, dan lainnya. Pada kalimat ini, kata Matsu diubah ke bentuk Te untuk membuat perintah. Untuk membuat perintah yang lebih halus, bisa dengan ditambahkan lagi kata Kudasai di belakang.

Kedua kata tersebut bila digabung akan menjadi ちょっと待って (Chotto matte), yang berarti "Tunggu bentar".

Bagaimana cara untuk mengubah suatu kata kerja ke bentuk Te?

Sebelumnya, perlu diketahui bahwa kata kerja Bahasa Jepang dibagi menjadi 3 kelompok. Masing-masing kelompok kata kerja memiliki ciri tersendiri, dan berbeda cara untuk dikonjugasikannya. Untuk mengubah KK ke bentuk Te, caranya tiap kelompok adalah sebagai berikut.

KK kelompok 1 : Untuk yang berakhiran 'u', 'tsu', 'ru', diubah menjadi 'tte'. Untuk yang berakhiran 'mu', 'nu', 'bu', diubah menjadi 'nde'. Untuk yang berakhiran 'ku', diubah menjadi 'ite'. Untuk yang berakhiran 'gu', diubah menjadi 'ide'. Untuk yang berakhiran 'su', diubah menjadi 'shιte'.KK kelompok 2 : Dengan mengubah akhiran 'ru' menjadi 'te'.KK kelompok 3 : Dengan mengubah 'suru' menjadi 'shιte', dan mengubah 'kuru' menjadi 'kite'.

Contoh :

Kau → KatteTatsu → TatteSuwaru → SuwatteYomu → YondeShinu → ShindeAsobu → AsondeAruku → AruiteOyogu → OyoideKesu → KeshιteMiru → MiteTaberu → TabeteBenkyousuru → BenkyoushιteMottekuru → Mottekite

Pola '∼te kudasai'

Fungsi dari pola ini adalah untuk membuat ungkapan permohonan atau perintah yang halus. Kata Kudasai ini memiliki arti "Mohon" atau "Tolong". Sedangkan 'te' di sini merupakan kata kerja dalam bentuk Te, yang fungsinya membuat suatu perintah.

Pola kalimat :

== [KK](Bentuk Te) + kudasai

Contoh :

Oshieru → Oshiete kudasaiYameru → Yamete kudasaiMatsu → Matte kudasai

7. Cara membuat matte smaleting


Matte adalah istilah yang digunakan dalam bidang pyrometalurgi yang diberikan pada fase logam sulfida cair yang biasanya terbentuk selama peleburan tembaga, nikel, dan logam tidak mulia lainnya. Biasanya, matte adalah fase di mana logam utama yang diekstraksi pulih sebelum proses reduksi akhir (biasanya mengkonversi) untuk menghasilkan logam mentah. Mattes juga dapat digunakan untuk mengumpulkan kotoran dari fase logam, seperti dalam kasus peleburan antimon. Mattes cair tidak larut dalam fase terak dan logam. Ketidaksuburan ini, dikombinasikan dengan perbedaan dalam gravitasi spesifik antara mattes, terak, dan logam, memungkinkan untuk pemisahan fase cair.

13. cara membuat matte semelting​


a. Tahap kominisi

Membebaskan atau meliberasi mineral-mineral tembaga dari ikatan zat-zat pengotornya dengan cara operasi peremukan dan penggerusan. Target dari Proses Pembuatan Tembaga ini ialah ukuran partikel bijih yang bisa memperoleh tingkat recoveri maksimal ketika tahap Konsentrasi Flotasi.

b. Tahab Konsentrasi Flotasi

Setelah ukurannya sesuai, dilanjutkan dengan tahap pemisahan mineral atau konsentrasi.Memisahkan mineral-mineral Cu-Fe-S dan Cu-S pengotoran yang cukup efektif dengan melakukan metode flotasi. Dengan menggunakan metode ini, bisa meningkatkan kadar tembaga di konsentrat menjadi 30 %.

c. Tahap Matte Smelting

Masuk ketahap ini konsentrat tembaga dileburkan berupa lelehan matte. Proses peleburan ini dalam keadaan oksidatif menghasilkan lelehan matte (sulfida yang kaya akan tembaga serta sedikit mengandung besi) dan slag (lelehan dari campuran oksida besi dan oksida logam pengotor serta fulk), gas buang. Tahap ini menghasilkan matte yang kandungan tembaganya setara 45% sampai 75%.

d. Tahap Konversi Matte

Tahap Konversi Matte menjadi blister copper, melalui dioksidasi dan kandungan meningkat menjadi 90 %.Concerter dihembuskan udara melewati sejumlah tuyeres yang terendam dalam lelehan. Dalam proses ini ditambakanpula oksigen murni, silika sebagai fliks, revert dan scrap. Slag yang akan dihasilkan memiliki senyawa besi –silika.

e. Tahap Fire refining,

proses ini biasa disebut pemurnian dilakukan dalam rotary-furnace, reverberatory-furnace, heart-furnace yang bisa diteliti dengan 2 tahap. Tahap pertama, oksidasi selektif kepada sulfur dan elemen pengotor. Tahap kedua, deoksidasi guna penurunan kandungan oksigen dalam tembaga.

Tahap fire refining di atas sanggup memperoleh logam tembaga yang mempunyai kandungan 99 %. Proses selanjutnya, melakukan pelarutan tembaga secara elektrokimia melalui tembaga anode dan mengendapkannya kembali di permukaan katode. Mineral-mineral pengotor yang terdapat pada tembaga anode mengapung ke permukaan dan tidak mengendap.Proses Pembuatan Tembaga yang terakhir ini ialah tahapan Elektrorefining.

14. Quizz for youBahasa JepangArti kata ganbatte kudasai dan chotto mattenote:​Cuman Quizz


頑張って下さい (ganbatte kudasai) : semangat ya, berjuanglah, semoga beruntung.ちょっと待って (chotto matte) : tunggu sebentar.

______________

Pembahasan

Ganbatte kudasai

「頑張って下さい・がんばってください」

= Semangat ya, berjuanglah, semoga beruntung.

Untuk pemakaian frasa ini tergolong formal, juga bersifat umum. Merupakan salah satu frasa yang digunakan sebagai ungkapan penyemangat atau memberikan motivasi agar dapat bersemangat kembali.

Frasa diatas merupakan bentuk frasa dari pola kata kerja -te + 下さい (kudasai).

Kata 頑張って (ganbatte) tergolong kata kerja, yang dimana awalnya kata kerja tersebut dituliskan 頑張る (ganbaru) artinya yaitu berjuang, semangat, berusaha.

Kemudian untuk kata 下さい (kudasai) ini dapat digunakan untuk meminta atau memohon kepada orang lain, dan juga bisa digunakan sebagai bentuk kata perintah dengan kesan sopan.

Kita juga bisa menggunakan kata 頑張って (ganbatte) saja sebagai kata penyemangat kepada orang lain, namun dengan konteks digunakan dalam situasi yang informal atau akrab.

--------

Chotto matte

「ちょっと待って・ちょっとまって」

= Tunggu sebentar.

Merupakan frasa yang digunakan dalam kondisi yang informal, sehingga tidak dapat digunakan dalam nuansa formal.

Kata ちょっと (chotto) artinya yaitu sedikit, bentar/sebentar. Kemudian untuk arti 待って (matte) adalah tunggu/menunggu, merupakan golongan kata kerja konjugasi -te. Kata kerja awalan 待って (matte) adalah 待つ (matsu).

Menurut Mattei, dalam masyarakat berkembang, seperti Indonesia, sistem hukum nasional yang bekerja adalah sistem hukum yang didominasi oleh... tradisi yang bersifat b Menurut Mattei , dalam masyarakat berkembang , seperti Indonesia , sistem hukum nasional yang bekerja adalah sistem hukum yang didominasi oleh ... tradisi yang bersifat b​


Dalam masyarakat berkembang, seperti Indonesia, sistem hukum nasional yang bekerja menurut pandangan Mattei adalah sistem hukum yang didominasi oleh karekteristik dari rule of profesional law.  Rule of profesional law lebih mendominasi sistem hukum di Indonesia dibandingkan dengan dua klafikasikasi sistem hukum lainnya menurut Mattei.

Pembahasan

Sistem hukum adalah suatu kesatuan yang terdiri dari unsur-unsur yang mempunyai interaksi satu sama lain dan bekerja sama untuk mencapai tujuan kesatuan tersebut. Kesatuan tersebut diterapkan terhadap kompleks unsur-unsur yuridis seperti peraturan hukum, asas hukum, dan pengertian hukum

Ugo Mattei salah satu pakar perbandingan hukum menawarkan taksonomi baru yang memperhitungkan sistem hukum negara-negara lain yang tidak dapat dikelompokkan ke dalam sistem hukum common fer atau civil law. Di dalam artikel "Three l'atterns of Law Taxonomy and Change in The World's Legal System". Ugo Mattei memandang perlunya membuat klasifikasi sistem hukum yang tidak tersentral pada klasifikasi Eropa Amerika (non-Euro-American-Centric classification). Keluarga hukum yang ditawarkan oleh Ugo Mettei yaitu keluarga hukum rule of profesional law, rule of political law, dan rule of tradisional law.

24. Sebutkan perbedaan kertas Silky paper dan kertas doff paper atau matte​


Jawaban:

HVS

Bahan kertasnya agak kasar, umumnya dipake untuk Fotocopy / Printer Deskjet. Kertas jenis ini banyak dijual di toko-toko buku.

Kertas jenis HVS dibuat dari pulp (bubur kertas) yang tidak mengandung lignin (perekat antar serat di dalam pohon), sehingga tidak mudah berubah warna (menjadi kekuningan) jika diletakkan di bawah sinar matahari atau sinar lampu.

Art Paper & Matt Paper

Bahan kertas untuk brosur, karena permukaannya yang licin (artpaper), atau yang semi doff (matt paper). Selain karena licin, hasil yang dihasilkan juga bagus, karena raster kertasnya halus.

Kertas art paper ini kurang bisa menyerap air sehingga kurang cocok digunakan dengan printer yang menggunakan tinta dye yang merupakan tinta berbahan dasar air.

Tinta yang cocok digunakan untuk mencetak dengan media kertas art paper ini adalah tinta art paper yang di desain khusus untuk head printer Epson.

Art Karton

Bahan kertas ini sama seperti art paper, cuma gramasinya lebih tebal. Banyak digunakan untuk cetakan seperti kartu nama, katalog, co profile,brosur, dan cetakan lainnya yang membutuhkan kertas agak tebal.

Umumnya setelah di cetak, bahan ini di lapisi laminating lagi (optional), supaya hasilnya lebih memuaskan. Sama seperti kertas art paper, kertas ini tidak bisa digunakan dengan tinta dye dan harus mengunakan tinta art paper yang di desain khusus untuk head printer Epson.

Matte / Doff paper

Sangat mudah menyerap tinta sehingga bisa dipakai untuk menyembunyikan kualitas tinta dan printer sekaligus tidak memantulkan cahaya.

Mencetak foto warna bisa dilihat dari kebutuhannya. Kalau untuk dibingkai atau banyak dipegang-pegang, kertas doff lebih awet dan tidak lengket. Kertas jenis ini seringkali menjadi rekomendasi untuk kertas cetak foto dengan hasil yang bagus.

Sublime Paper

Kertas jenis ini bukan digunakan untuk mencetak foto sebagai pajangan dirumah, didompet atau untuk dibingkai tetapi kertas ini digunakan sebagai mediator (media perantara) transfer gambar ke t-shirt (kaos).

Jadi bila kita ingin sebuah gambar dipindahkannya ke t-shirt (kaos) maka gunakanlah jenis Sublim Paper karena kertas ini mampu memindahkan tinta dengan maksimal ke t-shirt.

Double-Side Paper

Jenis kertas ini mampu digunakan untuk mencetak foto pada kedua sisinya (depan dan belakang). Kualitas foto yang dihasilkan juga cukup bagus, tidak terlalu mengkilap dan cenderung doff.

Jenis kertas ini cocok digunakan untuk mencetak pamflet yang biasanya digunakan untuk sarana promosi, sehingga para konsumen dapat melihat dikedua sisinya.

Fiber matte

Kertas Fiber Base paling tahan lama, karena dia menggunakan kertas dengan pH netral (biasa di sebut Archival Paper).

Premium Glossy foto Paper

Kertas jenis ini biasa disebut oleh para penggunanya dengan sebutan high glossy, kertas jenis ini mampu menghasilkan cetakan dengan efek yang lebih mengkilap.

Kertas jenis ini sangat cocok untuk mengcetak foto dengan resolusi tinggi. Walaupun harga kertas ini lebih mahal tetapi jika kita gunakan, akan menghasilkan cetakan foto yang maksimal dan lebih cerah.

Sticker Glossy foto Paper

Sering kita menjumpai sticker yang menampilkan foto dengan warna dasar kertas putih dan mengkilap, jenis ini sangat cocok untuk keperluan pembuatan sticker serta mampu mencetak foto beresolusi tinggi.

Laster foto Paper

Laster foto paper biasanya digunakan untuk keperluan dokumenter karena jenis kertas ini sangat awet bahkan bisa bertahan hingga puluhan tahun, tidak mudah pudar, mampu menghasilkan efek doff, dan sangat cocok untuk foto dengan resolusi tinggi.

34. Cara membuat matte smaleting


Matte adalah istilah yang digunakan dalam bidang pyrometalurgi yang diberikan pada fase logam sulfida cair yang biasanya terbentuk selama peleburan tembaga, nikel, dan logam tidak mulia lainnya. Biasanya, matte adalah fase di mana logam utama yang diekstraksi pulih sebelum proses reduksi akhir (biasanya mengkonversi) untuk menghasilkan logam mentah. Mattes juga dapat digunakan untuk mengumpulkan kotoran dari fase logam, seperti dalam kasus peleburan antimon. Mattes cair tidak larut dalam fase terak dan logam. Ketidaksuburan ini, dikombinasikan dengan perbedaan dalam gravitasi spesifik antara mattes, terak, dan logam, memungkinkan untuk pemisahan fase cair.
