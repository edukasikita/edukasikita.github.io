Dari konsep soal terlihat bahwa urutan bilangan mengalami pertambahan konsisten, dst. Konsep ini disebut dengan baris bilangan, atau baris aritmatika yaitu sebuah bilangan berurutan, dimana pada setiap bilangan selanjutnya mengalami penambahan dengan nilai sama pada bilangan sebelumnya.

rumus barisan arimatika, untuk mencari nilai ke n adalah

Un = a+(n-1)b

Un adalah urutan ke n

Dimana a adalah nilai awal atau bisa disebut dengan U1

Dan b adalah beda/penambahan bilangan tersebut

Dan untuk mencari jumlah dari suku barisan adalah

Sn = n/2(a+Un) atau Sn = (n/2)(2a + (n-1)b)

Sn = jumlah barisan

Berdasarkan soal diatas maka

a. 1,8,15,22,....92

a = 1

b = 4

Un = 92

karena yang ditanyakan adalah jumlah bilangan, maka kita cari nilai n nya dulu

maka

Un = a+(n-1)b

92=1+(n-1)4

92 = 1 +4n -4

92 = 4n -3

92+3 =4n

95 = 4n

n = 23,75

nilai n didapatkan maka kita bisa masukan ke persamaan nilai total jumlah bilangan

Sn = n/2(a+Un)

Sn = (23,75/2)(1+92)

Sn = 11,875 . 93

Sn = 1104375

Jadi total bilangan hingga baris ke 23,75 adalah 1104375

b. 7,11,15,19,....,103

a = 7

b = 4

Un = 103

nilai n didapatkan maka kita bisa masukan ke persamaan nilai total jumlah bilangan

Un = a+(n-1)b

103= 7 +(n-1)4

103 = 7 +4n-4

103 = 3+4n

100=4n

n = 25

nilai n didapatkan maka kita bisa masukan ke persamaan nilai total jumlah bilangan

Sn = n/2(a+Un)

Sn = (25/2)(7+103)

Sn = (25/2)110

Sn = 1375

Jadi total bilangan hingga baris ke 25 adalah 1375

c. 4,13,22,31,....,157

a= 4

b =9

Un = 157

nilai n didapatkan maka kita bisa masukan ke persamaan nilai total jumlah bilangan

Un = a+(n-1)b

157 = 4+(n-1)9

157 = 4+9n -9

157 = 9n-5

157+5 = 9n

162 = 9n

n = 18

nilai n didapatkan maka kita bisa masukan ke persamaan nilai total jumlah bilangan

Sn = n/2(a+Un)

Sn = (18/2)(4+157)

Sn = 9. 161

Sn = 1449

Jadi total bilangan hingga baris ke 18 adalah 1449

d. -8,-2,4,10,.....,106

a : -8

b = 6

nilai n didapatkan maka kita bisa masukan ke persamaan nilai total jumlah bilangan

Un = a+(n-1)b

106 = -8 +6n -6

106 = 6n -14

106 +14 = 6n

120 = 6n

31. Jika diketahui am=13cm, mn=29cm, dan ab=21cm, tentukan bn??


Jawaban:

7 cm

Penjelasan dengan langkah-langkah:

Soal Garis Singgung Dalam Dua Lingkaran

Rumus : SD² = JP² - (R+r)²

Dik:

Garis singgung dalam (SD) = AB = 21 cm

Jarak antara titik pusat (JP) = MN = 29 cm

Jari-jari lingkaran besar (R) = AM = 13 cm

Dit:

Jari-jari lingkaran kecil (r) = BN

SD² = JP² - (R+r)²

AB² = MN² - (AM + BN)²

21² = 29² - (13 + BN)²

441 = 841 - (13 + BN)²

(13 + BN)² = 841 - 441

(13 + BN)² = 400

(13 + BN) = √400

13 + BN = 20

BN = 20 - 13

BN = 7 cm

34. (an-¹bn)³=axby nilai dari y———— _a²nb⁶+n x​


Untuk menyelesaikan masalah ini, kita perlu melakukan beberapa langkah aljabar.

Pertama, kita dapat menulis ulang (an - bn)³ sebagai (a - b)³n³. Ini menggunakan rumus binomial untuk memperluas pangkat tiga.

Kedua, kita dapat menuliskan a²n sebagai (an)² dan b⁶ sebagai (b²)³.

Ketiga, kita dapat membagi baik pembilang dan penyebut dengan n³ sehingga kita dapat memindahkan pangkat n di dalam kurung.

Dengan melakukan langkah-langkah ini, kita dapat menuliskan ekspresi sebagai berikut:

(a - b)³ (an)² y

a²n² b³ + nx

Sekarang, karena kita ingin mengetahui nilai dari y, kita perlu memanipulasi persamaan sehingga y berada di sisi kiri.

Kita bisa mulai dengan mengalikan kedua sisi dengan a²n² b³ + nx untuk menghilangkan denominasi di bawah garis.

(a - b)³ (an)² y (a²n² b³ + nx)

--------------------------------- = y

a²n² b³ + nx

Ini memberikan kita ekspresi akhir:

y = (a - b)³ (an)²

a²n² b³ + nx

Sekarang kita memiliki persamaan untuk y dalam hal a, b, dan n.
