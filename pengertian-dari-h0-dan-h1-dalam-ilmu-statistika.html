Hipotesis H0 dan H1 yang tepat untuk menguji pernyataan tersebut adalah …. A. H0 : m10 dan H1 : m<10 B. H0 : m=10 dan H1 : m 10 C. H0 : m=10 dan H1 : m>10 D. H0 : m=10 dan H1 : m<10 E. H0 : m 10 dan H1 : m>10


Jawaban:

C. H0 : m=10 dan H1 : m>10

Penjelasan dengan langkah-langkah:

Dalam kasus ini, kita ingin menguji apakah rata-rata usia kulkas yang dimiliki oleh pengusaha es krim dapat bertahan lebih dari 10 tahun. Karena kita ingin menguji hipotesis mengenai rata-rata populasi, maka hipotesis nol (H0) harus merujuk pada nilai rata-rata tertentu, sedangkan hipotesis alternatif (H1) harus merujuk pada kemungkinan bahwa rata-rata populasi melebihi atau kurang dari nilai tertentu.

Dalam hal ini, karena kita ingin meneliti apakah rata-rata usia kulkas dapat bertahan lebih dari 10 tahun, maka hipotesis nol (H0) harus merujuk pada kasus ketika rata-rata usia kulkas sama dengan atau kurang dari 10 tahun. Sebaliknya, hipotesis alternatif (H1) harus merujuk pada kasus ketika rata-rata usia kulkas melebihi 10 tahun.

Dengan demikian, pilihan jawaban yang tepat adalah C.

H0 : rata-rata usia kulkas = 10 tahun

H1 : rata-rata usia kulkas > 10 tahun

Penjelasan Hipotesis:

pernyataan yg diajukan utk diuji kebenarannya melalui analisis data. hipotesis ini kemudian dibandingkan dgn data sebenarnya untuk menyimpulkan apakah hipotesis tersebut diterima atau ditolak.

Jenis Hipotesis:

1. Hipotesis Nol

pernyataan yg menyatakan bahwa tidak ada perbedaan, hubungan, atau efek dri 2 atau lebih variabel pernyataan ini biasanya dibuat secara default dan diperlakukan sebagai "tidak ada efek" atau "tidak ada perbedaan".

2.  Hipotesis Alternatif

pernyataan yg menyatakan adanya perbedaan, hubungan, efek dri 2 atau lebih variabel.

3. Hipotesis Alternatif Satu Arah

menyatakan perbedaan atau efek dlm satu arah tertentu.

4. Alternatif Dua Arah:

menyatakan adanya perbedaan atau efek dlm kedua arah.

Rumus:

Z-score:

Z = (X - μ) / σ

t-score:

t = (X - μ) / (s / √n)

Z = nilai Z-score atau t-score.

X = rata" sampel atau populasi.

μ = rata" populasi.

σ = deviasi standar populasi.

s = deviasi standar sampel.

n = ukuran sampel.

Contoh:

H0 (hipotesis nol) = pendapatan buruh rata" tidak = rp800.000,00 per bulan

H1 (hipotesis alternatif) = pendapatan buruh rata" = rp800.000,00 per bulan

36. Sebuah bola dijatuhkan dari ketinggian 100 cm di atas lantai. maka tinggi pantulan kedua, ketiga dan keempat bola adalah...diket : h0 = 100cmh1 = 60cm


Maka, tinggi pantulan

kedua = 0,36 meter ketiga = 0,216 meterkeempat = 0,1296 meter

Penjelasan dengan langkah langkah

Diketahui

[tex]ho \ = 100 \ cm \ = 1 \m \\\\ h1 \ = 60 \ cm \ = 0,6 \ cm[/tex]

Ditanya

maka tinggi pantulan kedua, ketiga dan keempat bola adalah...

Dijawab

Menentukan e terlebih dahulu

[tex]e \ = \sqrt{\frac{h1}{h0} } \\\\ e \ = \sqrt{\frac{0,6}{1} } \\\\ e \ = \sqrt{0,6} \\\\ e \ = \frac{\sqrt{15} }{5}[/tex]

Menentukan tinggi pantulan kedua

[tex]e \ = \sqrt{\frac{h1}{h0} } \\\\ e\ = \frac{\sqrt{15} }5} \\\\ \frac{\sqrt{15} }5} \ = \sqrt{\frac{h2}{h1} } \\\\ (\frac{\sqrt{15} }5})^2 \ = (\sqrt{\frac{h2}{0,6} } )^2 \\\\ \frac{15}{25} \ \ = \frac{h2}{0,6 } \\\\ h2 \ = 0,36 \ meter[/tex]

Menentukan tinggi pantulan ketiga

[tex]e \ = \sqrt{\frac{h1}{h0} } \\\\ e\ = \frac{\sqrt{15} }5} \\\\ \frac{\sqrt{15} }5} \ = \sqrt{\frac{h3}{h2} } \\\\ (\frac{\sqrt{15} }5})^2 \ = (\sqrt{\frac{h3}{0,36} } )^2 \\\\ \frac{15}{25} \ \ = \frac{h3}{0,36 } \\\\ h3 \ = 0,216 \ meter[/tex]

Menentukan tinggi pantulan keempat

[tex]e \ = \sqrt{\frac{h1}{h0} } \\\\ e\ = \frac{\sqrt{15} }5} \\\\ \frac{\sqrt{15} }5} \ = \sqrt{\frac{h4}{h3} } \\\\ (\frac{\sqrt{15} }5})^2 \ = (\sqrt{\frac{h4}{0,216} } )^2 \\\\ \frac{15}{25} \ \ = \frac{h3}{0,216 } \\\\ h4 \ = 0,1296 \ meter[/tex]
