4. pengertian stability of value?


Stability of Value artinya mempunyai nilai yang yang stabil. Nilai yang stabil maksudnya adalah nilai yang relatif tetap dan tidak mudah berubah.
Stability of value artinya memiliki nilai tetap, yang dimaksudkan yaitu nilai uang di masa ini memiliki nilai sama di masa mendatang sehingga masyarakat percaya menyimpan uang tidak akan dirugikan.

14. rumus Present Value ?


FV . PFVIF (i,n)

dengan:

r = tingkat bunga per periode waktunya = jumlah tahunPV = jumlah pokok pada waktu sekarang
(present time)FV = future (compound) value pada akhir tahun ke-nPFIV = present value interest factor

Rumus present value :

= FV . PFVIF (i,n)

keterangan :
r => tingkat bunga per periode waktu
n => jumlah tahun
PV => jumlah pokok pada waktu sekarang (present time)
FV => future (compound) value pada akhir tahun ke-n
PFIV => present value interest factor

20. Jelaskan perbedaan Present Value (PV) dengan Future Value (FV) dan berikan contoh simulasi perhitungannya!


Jawaban:

Perbedaan Present Value (PV) dengan Future Value (FV) dan contoh simulasi perhitungannya yaitu jika present value menunjukkan nilai uang saat ini dan future value menunjukkan nilai uang dimasa yang akan datang dan menggambarkan manfaat dari sebuah investasi.

Contohnya dapat dilihat pada penjelasan dibawah ini.

Penjelasan:

Present value: Anda memiliki uang saat ini Rp.2.000.000 jika anda menginvestasikannya selama 5 tahun dengan suku bunga 5% maka jumlah yang dimiliki adalah?

Present value yaitu jumlah uang saat ini sebesar Rp.2.000.000

Future value :

Future value = PV x (1+i)n

PV : Present value

i : suku bunga

n: jangka waktu/periode

Future value = PV x (1+i)n

                      = Rp.2.000.000 x (1 + 5%) 5

                      = Rp.2.000.000 x [1+(0,005 x 5)]

                      =  Rp.2.050.000

Maka besarnya future value adalah sebesar Rp.2.050.000

33. cara mengerjakan menghitung present value jika obligasi 100 dengan bunga 9%​


Diketahui

Nilai pokok obligasi = 900Bunga = 9%Jumlah tahun = ..?

Ditanyakan

PV = ...?

Penjelasan

Oke, untuk menghitung present value kita harus mengetahui present value interest factor nya dulu dengan rumus :

PVIF = 1 / (1 + r)^n

Namun, karena disoal tidak diketahui jumlah tahunnya, maka cobalah kita anggap hanya 1 tahun :

PVIF = 1 / (1 + r)^n

PVIF = 1 / (1 + 9%)^1

PVIF = 1 / (1 + 0,09)^1

PVIF = 1 / (1 + 0,09)

PVIF = 1 / 1,09

PVIF = 0,91743

Maka, nilai PV dapat kita hitung dengan mengalikan PVIF dengan obligasinya :

PV = PVIF × O

PV = 0,91743 × 100

PV = 91,743 (kurang lebih)
